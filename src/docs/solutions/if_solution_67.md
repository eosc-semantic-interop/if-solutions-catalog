# Uniform Data Access Layer

|Type|Status|
|----|------|
|Standard|In development|

Problem ID(s) this solutions adresses:
- T2

## Description
Embraces current diversity and future technical innovation in specific and tuned data management - unifies them from a client side data access point of view. Applies a design by contract approach to guarantee sustainability.

## Problems that CAN be solved with this IF solution
Have durable (sustainable) VRE solutions, reporting pipelines, and guaranteed historic data production systems in world with changing technical stacks for data management and disclosure between the two.

## Limitations of this IF solution
currently only python support

## Description of use case/scenario/case of study for this IF solution
- I want to feed my report/ analysis/ modal with quality data without needing to know where and how it is maintained
